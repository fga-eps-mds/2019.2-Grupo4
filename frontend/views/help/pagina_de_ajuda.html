Sobre o Questmark

O que é?

Questmark é uma plataforma online para a criação de questionários, onde você escreve em Markdown!

Os questionários criados podem ser compartilhados para que sejam respondido e, no final, você terá acesso a uma dashboard com as respostas e suas estatísticas!

Uso

  • Para criar um campo de texto : name = ___ OU name = ___[50]

  • Para adicionar um 'radio button' : sex = () male () female

  • Para adicionar uma Checkbox : phones = [] Android [x] iPhone [x] Blackberry

  • Para adicionar um dropdown : city = {BOS, SFO, (NYC)}

  • Para definir os campos obrigatórios, basta inserir um '*' : zip code* = ___
